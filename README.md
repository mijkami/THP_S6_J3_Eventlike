### Eventbrite-like Project

  
## How to setup :

~~~~~~~~~~~~~~~~~
$ bundle install

$ rails db:create

$ rails db:environment:set RAILS_ENV=development

$ rails db:schema:load

$ rails db:seed

$ rails console
~~~~~~~~~~~~~~~~~
